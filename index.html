Fundamentos de HTML y CSS

Introducción a HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Define la estructura y el contenido de una página web utilizando una serie de elementos y etiquetas.

  • <html>: Elemento raíz de la página.
  • <head>: Contiene metadatos y enlaces a recursos externos.
  • <body>: Contiene el contenido visible de la página.
  • <h1> a <h6>: Encabezados de diferentes niveles.
  • <p>: Párrafos de texto.
  • <a>: Enlaces (hipervínculos).
  • <img>: Imágenes.
  • <ul>, <ol>, <li>: Listas.

Introducción a CSS

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo utilizado para describir la presentación de un documento escrito en HTML. Permite controlar colores, fuentes, diseño y más.

  • Selectores: Apuntan a elementos HTML para aplicar estilos (ej. p, .clase, #id).
  • Propiedades: Características que se desean cambiar (ej. color, font-size, margin).
  • Valores: El ajuste específico para una propiedad (ej. red, 16px, 20px auto).
  • Modelo de Caja: Cada elemento HTML es una caja con contenido, padding, borde y margen.
  • Flexbox y Grid: Sistemas de diseño avanzados para crear layouts complejos y responsivos.

Elementos HTML Comunes

Algunos de los elementos HTML más utilizados para estructurar contenido:

<div>Contenedor genérico</div>
<span>Contenedor en línea</span>
<form>Formularios</form>
<input type="text">Campos de entrada</input>
<button>Botones</button>

Propiedades CSS Esenciales

Propiedades CSS fundamentales para estilizar tus páginas:

body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
}
.bento-item {
    background-color: white;
    border-radius: 8px;
    padding: 20px;
    box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}

Diseño Responsivo

El diseño responsivo asegura que tu sitio web se vea bien en todos los dispositivos. Se logra principalmente con:

  • Media Queries: Permiten aplicar estilos diferentes según el tamaño de la pantalla.
  • Unidades relativas: Como %, em, rem, vw, vh.
  • Flexbox y Grid: Facilitan la adaptación del layout.
